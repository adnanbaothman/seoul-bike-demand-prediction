## 🚀 Live Demo

[Open the App](https://huggingface.co/spaces/AdnanBaothman/seoul-bike-demand-intelligence)

## Overview

Seoul Bike Demand Intelligence is an end-to-end machine learning dashboard for predicting hourly bike rental demand using weather, seasonality, and operational factors from the Seoul Bike Sharing Demand dataset.

This project turns a notebook experiment into a deployed portfolio product with:

- Interactive demand prediction
- Operational insights dashboard
- Machine learning model comparison
- Feature importance analysis
- CSV data exploration and export
- Hugging Face Spaces deployment

## Tech Stack

- Python
- Streamlit
- Pandas / NumPy
- Scikit-learn
- Plotly
- Hugging Face Spaces
